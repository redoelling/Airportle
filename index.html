Boarding Pass

Every day, four flights are waiting. Your job: identify each 3-letter IATA code in 6 attempts. Type any code from the list and the board lights up to tell you how close you are.

Heads up: Airportle uses a hand-picked subset of the world's IATA codes, not all ~11,300 of them. Only codes that live in this dataset count as valid guesses — if a real airport gets rejected, it just isn't in the deck.
